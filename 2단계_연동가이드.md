## 1. `core/llm.py` — OpenAI 함수 4개 채우기

파일 상단에 이걸 추가:

```python
import streamlit as st
from openai import OpenAI

@st.cache_resource
def _client():
    """클라이언트는 한 번만 만들어 재사용 — st.cache_resource 로 캐싱"""
    return OpenAI(api_key=st.secrets["OPENAI_API_KEY"])
```

### 1-1. `analyze_meeting()` — 노트북 미션 3

mock 부분을 지우고 이렇게 교체:

```python
def analyze_meeting(text: str, meeting_id: str = "tmp") -> AnalysisResult:
    sys_role = """
    당신은 '김 대리'의 개인 비서 '에이블'입니다.
    아래 회의 내용을 읽고 다음 4가지를 JSON 형식으로 정리해주세요:
    - summary: 한 문단 요약
    - decisions: 주요 결정사항 글머리
    - personal_impact: 김 대리가 처리해야 할 업무 [{title, due (YYYY-MM-DD), quadrant (Q1~Q4)}]
    - open_questions: 추가 확인이 필요한 사항
    """
    prompt = f"[미팅 내용]\n{text}"

    response = _client().chat.completions.create(
        model="gpt-4o-mini",
        messages=[
            {"role": "system", "content": sys_role},
            {"role": "user", "content": prompt},
        ],
        response_format={"type": "json_object"},  # JSON 강제
    )
    import json
    data = json.loads(response.choices[0].message.content)

    # due 문자열을 date 객체로 변환
    from datetime import datetime
    for item in data.get("personal_impact", []):
        if isinstance(item.get("due"), str):
            try:
                item["due"] = datetime.strptime(item["due"], "%Y-%m-%d").date()
            except ValueError:
                item["due"] = None

    return AnalysisResult(
        meeting_id=meeting_id,
        summary=data.get("summary", ""),
        decisions=data.get("decisions", []),
        personal_impact=data.get("personal_impact", []),
        open_questions=data.get("open_questions", []),
    )
```

### 1-2. `stt()` — 노트북 미션 4 (Whisper)

```python
def stt(audio_bytes: bytes) -> str:
    import io
    f = io.BytesIO(audio_bytes)
    f.name = "recording.m4a"  # OpenAI는 파일명 확장자로 타입 판단

    transcript = _client().audio.transcriptions.create(
        model="whisper-1",
        file=f,
    )
    return transcript.text
```

### 1-3. `tts()` — 노트북 미션 5

```python
def tts(text: str, voice: str = "alloy") -> bytes:
    speech = _client().audio.speech.create(
        model="tts-1",
        voice=voice,
        input=text,
    )
    return speech.read()
```

## 4. 추가로 챙기면 좋은 것

### 4-1. `@st.cache_data` 로 LLM 응답 캐싱

같은 회의를 여러 번 분석하지 않도록:

```python
@st.cache_data(show_spinner=False)
def _cached_analyze(text: str) -> dict:
    # ... LLM 호출 ...
    return {...}
```

### 4-2. 에러 처리 강화

```python
try:
    result = llm.analyze_meeting(text)
except Exception as e:
    st.error(f"분석 실패: {e}")
    st.stop()
```

### 4-3. 비용 관리

- `gpt-4o-mini` 우선 사용 (저렴)
- 텍스트가 너무 길면 LLM에 보내기 전에 길이 제한 (예: 8000자)
- TTS는 텍스트 길이만큼 과금되므로 요약 결과만 음성화

### 4-4. 영구 저장 (선택)

session_state는 브라우저 새로고침하면 사라져. 영구 저장하려면 `core/store.py`의 `add_todo` 같은 함수에 SQLite 쓰기 한 줄만 추가하면 됨. (스키마는 `core/models.py` 그대로)

```python
# core/store.py 예시
import sqlite3
DB = sqlite3.connect("able.db", check_same_thread=False)
# init: CREATE TABLE IF NOT EXISTS todos(...)
def add_todo(...):
    t = Todo(...)
    DB.execute("INSERT INTO todos VALUES (?, ?, ...)", (t.id, t.title, ...))
    DB.commit()
    st.session_state.todos.append(t)
```

## 5. 자주 막히는 점

- **`st.secrets` 가 비어 있다고 에러** → `.streamlit/secrets.toml` 위치 확인. `app.py` 와 같은 폴더의 `.streamlit/` 안에 있어야 함.
- **JSON 파싱 실패** → LLM이 가끔 JSON 외 텍스트를 같이 뱉음. `response_format={"type":"json_object"}` 옵션을 꼭 켤 것.
- **녹음 파일 업로드 시 MIME 에러** → `f.name = "rec.m4a"` 처럼 OpenAI가 인식할 확장자 부여.
- **TTS 응답이 너무 큰 경우** → 텍스트를 1000자 이내로 자르고, 필요하면 분할 합성.
